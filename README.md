#Projeto App Banking criado com React Native e Typecript

Para rodar a aplicação são necessários a instalação:
- Android Studio (para testar android)
- Xcode (para testar ios)
- Node versão mais recente
- Yarn ou NPM

Para iniciar o emulador rodar o comando:
- android:  "npx react-native run-android"
- ios: "npx react-native run-ios"
